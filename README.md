# 🚀 SECOP Smart

SECOP Smart es una plataforma desarrollada para automatizar la consulta, análisis y gestión de procesos de contratación publicados en **SECOP II**, facilitando la búsqueda de oportunidades y la generación de documentos para la presentación de ofertas.

---

## 📌 Objetivo

Reducir el tiempo invertido en la búsqueda y análisis de procesos de contratación mediante la automatización de tareas repetitivas.

Entre sus funcionalidades se encuentran:

- Consulta de procesos en SECOP II.
- Filtros avanzados de búsqueda.
- Exportación de resultados a Excel.
- Análisis de documentos del proceso.
- Generación automática de documentos en Word.
- Seguimiento de procesos y ofertas.
